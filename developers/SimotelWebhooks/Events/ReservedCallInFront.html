ReservedCallInFront

The ReservedCallInFront event is triggered after a call is created in the specified queue where the call reservation feature is enabled from the queue component. This event will be generated and sent to your web service.

Event Parameters​

DescriptionDefined DataSample DataParameters
Event Name-ReservedCallInFrontevent_name
Event Type (same as name)-ReservedCallInFrontevent
Name of the queue where the call is placed-1010queue_name
Caller’s phone number (Caller ID)-09123456789caller_id
Date and time the call entered the queue (Gregorian - YYYY-MM-DD HH:MM:SS.mmmmmm)-2025-03-11 13:10:12.712188date
