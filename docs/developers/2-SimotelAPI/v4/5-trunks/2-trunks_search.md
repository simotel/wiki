# Trunk Search

This service is used to search for trunks.

## Parameters
<div class="custom-table">
|       Description      | Defined Data  | Sample Data    | Required[**]/Optional[*] |     Parameters    |
|:----------------------:|:--------------:|:--------------:|:-------------------------:|:------------------:|
| View Similar Items     |     true/false |      true      |           **              |       alike        |
|      Trunk Name        |        -       | trunk_name_new |                          | conditions->name   |
</div>
