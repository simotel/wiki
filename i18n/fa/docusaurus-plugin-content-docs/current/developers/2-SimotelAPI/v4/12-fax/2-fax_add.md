# Fax Add

از این سرویس برای افزودن فکس ارسالی استفاده می‌شود.

:::tip نکته
برای ارسال فکس به صورت فایل فیلد "file" باید مقدار داشته باشد و فیلد "text" از JSON ورودی حذف شود. 
:::
